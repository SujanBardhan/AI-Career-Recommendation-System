# AI Career Recommendation System

## Overview
This project is an AI-powered Career Recommendation System that helps users discover suitable career paths based on their interests and preferences. It uses Google Gemini AI to generate personalized career recommendations.

## Features
- Career assessment quiz
- AI-generated career recommendations
- Clean and responsive user interface
- Built using Flask (Python)
- Uses Google Gemini API

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Google Gemini AI

## Installation

1. Clone the repository:
```
git clone https://github.com/SujanBardhan/AI-Career-Recommendation-System.git
```

2. Install dependencies:
```
pip install -r requirements.txt
```

3. Run the application:
```
python app.py
```

4. Open your browser:
```
http://127.0.0.1:5000
```
